This is a small program that give 3 inital 
values and the pipline meters' list.
meter list rule:
	'1': there is 1 subline meter
	'2': there is 2 subline meters
	'0': there is 1 main line meter
ex: 10112110: 
	1: main meter; 
	0: 1 main line meter
	1: 1 subline meter
	1: 1 subline meter
	2: 2 subline meter
	1: 1 subline meter
	1: 1 subline meter
	0: 1 main line meter
